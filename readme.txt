Displays tornado warnings from the National Weather Service on your wordpress blog.

== Description ==

This plugin displays tornado warnings in the United States on your wordpress blog. 

Note: This plugin makes calls to a remote server to gather the latest tornado warning data.

== Installation ==

1. Upload the plugin

2. Activate Plugin

3. Put the tornado warnings widget where you want the tornado warnings to display

== Frequently asked questions ==

= Do you have tornado warnings for outside the United States? =
Not currently.
